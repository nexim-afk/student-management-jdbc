## About This Project
This project demonstrates how a Java application communicates 
with a MySQL database using JDBC (Java Database Connectivity).

## How It Works
1. MySQL database 'college' is created with a students table
2. Table stores student id, name and marks
3. Java code establishes connection to MySQL using JDBC driver
4. INSERT query adds student data into the table
5. Exception handling catches any connection or query errors

## What I Learned
- How to set up a MySQL database and create tables
- How JDBC connects Java to a database
- How to write and execute SQL queries from Java
- Importance of exception handling in database operations
- Difference between Statement and Connection objects in JDBC
